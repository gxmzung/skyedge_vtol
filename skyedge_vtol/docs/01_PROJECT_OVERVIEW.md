# 프로젝트 개요

## 목표
9월 대회 전까지 PX4 기반 VTOL 구조 임무 소프트웨어를 완성한다.

## 원칙
- FC는 PX4를 사용한다.
- 대회 핵심 점수와 연결되는 미션 상태머신, 비전, 가이던스, 실패 대응은 자체개발한다.
- 시뮬 성공만으로 완료로 보지 않는다.
- 실기체 통합 전까지 소프트웨어 불확실성을 최대한 제거한다.

## 시스템 계층
1. FC / PX4 계층
2. PX4 Bridge 계층
3. Mission / Guidance / Vision 계층
4. 대회 운영 / 로그 / PM 계층

## 대회 임무 흐름
이륙 → 탐색 → 목표 확인 → 정렬 → 구조 동작 → 복귀 → 착륙
